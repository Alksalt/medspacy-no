# Third-Party Sources and Provenance

This project uses third-party work as citations, seed material, and ecosystem dependencies. Production Norwegian clinical rules remain owner-reviewed project work, not automated translation output.

## Seed and Prior-Art Sources

- Skeppstedt 2011 Swedish NegEx triggers: used as linguistic seed material and cited prior art. The source article is CC BY 2.0; copied trigger lists or translated derivatives require attribution.
- Norwegian Centre for E-health Research 2018 poster: used as prior-art evidence and rule-count guidance. No reusable Norwegian lexicon file is bundled.
- medspaCy: MIT-licensed dependency and upstream target.
- spaCy Norwegian Bokmal models: installed separately by users; not bundled in this package.

## Norwegian Data Resources

- NorSynthClinical and NorSynthClinical-PHI: no explicit repo license found during scaffold review; do not redistribute derivative sentences without written permission.
- NorMedTerm: README currently states CC BY 4.0. Attribution is required, and bundling remains out of scope for v0.1.
